# Assets Directory

This directory contains all game assets organized by type.

## Structure

### models/
3D model files (.obj, .gltf, etc.)
- Player models (ball, frog rig)
- Platform models
- Environment objects (walls, barriers)
- Interactive objects (buttons, springs, bugs)

### textures/
Texture image files for materials.
- Diffuse maps
- Normal maps
- Roughness maps
- Emissive maps

### materials/
Godot material resources (.tres).
- Shared material definitions

### audio/
Audio files organized by type.

#### sfx/
Sound effects.
- Button press sounds
- Bug collection sounds
- Spring activation sounds
- Ball rolling sounds

#### music/
Background music tracks.

### animations/
Animation files.
- Aro (frog) character animations
- Object animations (button press, spring bounce)
- UI animations

### ui/
UI asset files.
- Icons
- UI textures
- Button graphics
- Menu backgrounds
