# Scenes Directory

This directory contains all Godot scene files (.tscn) organized by type.

## Structure

### game_objects/
Prefabs and reusable game object scenes.
- player.tscn - Main player prefab with ball and frog rig
- walls.tscn - Wall obstacle prefab
- button.tscn - Interactive button prefab
- barrier.tscn - Barrier/gate prefab
- goal_button.tscn - Level completion goal prefab
- bug.tscn - Collectible bug prefab

### levels/
Level scene files.
- test_world - Test level for testing game objects and interactions

### ui/
User interface scenes.
- main_menu.tscn - Main menu screen
- pause_menu.tscn - Pause menu overlay
- hud.tscn - In-game HUD overlay
- settings_menu.tscn - Settings menu screen
